二、版本号说明：
	例如：
		读出的版本是 V 20.01.303( 表示软件版本为003 )
	具体说明：
		第0位：2
		第1位：0        20：代表是舵机固件
		第2位：.
		第3位：0
		第4位：1        代表协议类型: 01：firmata协议        02：gcode协议
		第5位：.
		第6位：3        代表舵机种类标号 范围(0-9 A-Z a-z)
		第7位：0
		第8位：3
